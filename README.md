## Commands

-   Bundle install ... gets all gems

-   rails db:reset ... reset db

-   rails db:setup ... reset + seed

-   rails db:seed ... runs seeds.rb

-   rails db:migrate ... migrates

-   rubocop ... lints

-   rubocop -A ... corrects linting issues

-   rspec ... runs tests
